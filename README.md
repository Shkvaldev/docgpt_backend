# GPT Docs backend
Backend for project that generates documents using AI.

All project's info can be obtained through the documentation.

## Documentation

To access docs you need to do this:
```bash
git clone <repo>
cd <project name>
pip install -r requirements-docs.txt
mkdocs serve
```
